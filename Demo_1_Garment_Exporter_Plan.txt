TECH STACK
================================================================================

  Framework: Next.js 14+ (App Router)
  Styling: Tailwind CSS
  Language: TypeScript
  Animations: Framer Motion (subtle, professional — NOT flashy)
  Icons: Lucide React
  Images: Unsplash / Pexels (garment/textile/factory images)
  Fonts: Inter (body) + Playfair Display (headings) — professional look
  Deployment: Vercel (free)
  CMS: None (static for demo — keeps it fast)

================================================================================
DESIGN GUIDELINES
================================================================================

Colors:
  Primary: Deep navy blue (#1B2A4A) — trust, professionalism
  Secondary: Gold/amber (#C9A84C) — premium feel
  Accent: White (#FFFFFF) — clean
  Text: Dark gray (#333333) — readable
  Background: Light gray (#F8F9FA) — sections alternate white/light gray
  CTA buttons: Gold (#C9A84C) or Green (#25D366 for WhatsApp)

Typography:
  Headings: Playfair Display (serif) — premium, established feel
  Body: Inter (sans-serif) — clean, modern, readable
  Sizes: H1=48px, H2=36px, H3=24px, Body=16px

Layout:
  - Max width: 1280px, centered
  - Generous whitespace — don't crowd elements
  - Professional, not flashy — this is B2B, not a gaming site
  - Every section should breathe

Images:
  - Use high-quality Unsplash images:
    - Search: "t-shirt flat lay", "garment factory", "textile",
      "cotton fabric", "polo shirt", "knitwear"
    - Download 1200x800 or larger
    - Optimize for web (use next/image)
  - Hero: Dark factory/garment image with text overlay
  - Products: Clean, white-background product photos

Mobile:
  - MUST be perfect on mobile — many buyers check on phone first
  - Hamburger menu on mobile
  - Sticky WhatsApp button
  - Touch-friendly buttons (min 44px tap target)

Animations (Framer Motion):
  - Subtle fade-in on scroll for sections
  - Smooth page transitions
  - Product card hover: slight lift + shadow
  - Hero text: slide-in from left
  - Stats: count-up animation
  - DO NOT: flashy animations, auto-playing carousels, blinking elements

Performance:
  - Target: 90+ Lighthouse score
  - Use next/image for all images
  - Lazy load below-fold images
  - Minimize JS bundle
